**1. What is JSX, and why is it used?**<br>
**Ans:**
JSX is **JavaScript XML**.JSX is a way to write HTML code inside **JavaScript**. It helps you build UI easily and makes the code more readable.<br>
**2. What is the difference between State and Props?**<br>  
**Ans:**
**State** is the internal data of a component that can change over time. Props are data passed from a parent component to a child component and cannot be changed by the child.<br>
**3. What is the useState hook, and how does it work?**<br>  
**Ans:**
**useState** is a **React hook that allows functional components** to have state. It returns a state value and a function to update it. When the state changes, the component re-renders.<br>
**4. How can you share state between components in React?**<br>  
**Ans:**
**State** can be shared by moving it to a parent component and passing it as props, using **Context API** for global state, or using state management libraries like Redux.<br>
**5. How is event handling done in React?**<br>  
**Ans:**
**React** uses events like onClick, onChange, etc. You pass a function as the event handler. Events are written in **camelCase**.<br>
